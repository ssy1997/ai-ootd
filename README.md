# AI OOTD - 智能穿搭管理助手

AI OOTD 是一款基于微信小程序云开发和多模态大模型的智能衣橱管理与穿搭推荐应用。它通过 AI 技术帮助用户数字化管理衣橱资产，实现“越用越懂你”的个性化穿搭建议。

## ✨ 核心功能

- **智能服饰识别**：采用多模态 Vision 模型，支持一图分拣多件单品，自动提取品类、款式、材质等 8 维度属性。
- **习惯学习机制**：AI 在识别时会参考用户已有的标签库进行归一化打标，拒绝“长裤”与“裤子”等同义词污染。
- **对话式 AI 推荐**：结合实时天气、场合需求与衣橱库存，提供极具“时尚感”的穿搭方案及专业点评。
- **风格偏好画像**：基于用户真实行为（收藏/标记穿过）自动进化，可视化展现个人风格 DNA。
- **数字衣橱管理**：提供资产看板，动态统计单品利用率，辅助用户进行高效的穿搭决策。

## 🛠️ 技术架构

- **前端**：微信小程序原生框架 (TypeScript + SCSS)
- **渲染引擎**：Skyline 渲染引擎 + glass-easel 框架
- **后端**：微信云开发 (Cloud Functions, Cloud DB, Cloud Storage)
- **AI 推理**：字节跳动 Ark (豆包) 多模态大模型
- **外部集成**：和风天气 API（实时环境感知）

## 🎨 设计规范：Frosted Chic (磨砂时尚)

- **配色方案**：系统底色 `#F2F2F7` (Cool Gray)，品牌高亮色 `#7873f5` (Lavender Accent)。
- **交互形态**：全面采用 **100rpx 大圆角（胶囊化）** 标签与按钮，配合 **32rpx 卡片圆角**，追求极简与轻盈感。
- **视觉对齐**：全站图标采用 CSS 伪元素物理绘图，确保几何级绝对居中。

## 🚀 快速启动

1.  **环境准备**：克隆项目并在微信开发者工具中导入。
2.  **云开发设置**：开通云开发环境，并按 `cloudfunctions/database-init` 逻辑初始化集合。
3.  **密钥配置**：在 `cloudfunctions/ai/.env` 中配置你的 `ARK_API_KEY`。
4.  **编译预览**：点击编译，开启你的智能穿搭之旅。
